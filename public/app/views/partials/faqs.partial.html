FAQ's

Q: HOW TO REQUEST FOR A NEW TOOL?

Please follow the steps below:
  1. From the navigation pane, click Request New Tool and fill up the necessary details.
  2. Inside the Project Form, click App Actions then Create CR.
  3. Indicate "New Automation Tool" in the Request Summary field.
  4. Click Submit.

Q: WHERE CAN I SEE MY CHANGE REQUESTS / PROBLEM REPORTS?

You can see them from the left navigation pane under My Views. You can also opt to view all change requests and/or problem reports that you have access to from the Summary Views

HOW DO I MAKE A NEW CHANGE REQUEST / PROBLEM REPORT?

There are two ways to create a CR / PR:

Method #1: From the navigation pane

  1. Click either File Enhancement (CR) button or Raise Tool Issue (PR) button.
  2. A dialog box will appear containing a view of all the IIA apps/tools.
  3. Select a project document from the view. This will serve as the basis for your request/report.
  4. Click OK.

Method #2: Within the Project Form

  1. Click App Actions then click either Create CR or Create PR.

Q: IN PROJECT FORM, WHO SHOULD BE THE BUSINESS OWNER?

Business Owner should be the person who will take ownership of the new tool that you are requesting. For more details, please reach out to Mark Andrew L San Miguel/Philippines/IBM or Ralph John L Ramirez/Philippines/IBM.

Q: IN PROJECT FORM, I DON'T KNOW WHO SHOULD BE THE DEVELOPER AND/OR SUPPORT FOR THE NEW TOOL. WHAT SHOULD I DO?

Click App Actions then Set Default Names to populate with default values.
